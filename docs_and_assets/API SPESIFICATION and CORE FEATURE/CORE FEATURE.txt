1. Authentication & User Management

Register & Login (JWT based) untuk mahasiswa & UMKM.

Role system (Mahasiswa, UMKM, Admin) → biar hak akses beda.

Profile management (update bio, skill, portofolio mahasiswa / profil bisnis UMKM).




2. Project Marketplace

Posting Project (UMKM bisa upload kebutuhan, budget, deadline, kategori).

Project Browsing & Search (Mahasiswa bisa filter project berdasarkan skill, kategori, budget).

Apply/Proposal System (Mahasiswa bisa apply dengan penjelasan singkat).

Project Status Flow → OPEN → ONGOING → DONE → CLOSED.



3. Matching & Contract

Project Matching → mahasiswa pilih proyek sesuai skill.

Kontrak Digital Sederhana → semacam “agreement” dalam sistem saat UMKM memilih mahasiswa.



4. Communication System

Realtime Chat → mahasiswa & UMKM bisa diskusi detail project.

Opsional: attachment upload (kirim gambar wireframe, dokumen requirement, dsb).


5. Payment & Transaction

Payment Gateway Integration (Midtrans/Xendit/GoPay/DANA).

Sistem escrow sederhana (UMKM bayar → dana ditahan → cair kalau project selesai).



6. Review & Rating

UMKM kasih rating + review mahasiswa setelah project.

Mahasiswa juga bisa nilai pengalaman dengan UMKM (two-way rating).







7. Admin Panel

Moderasi user & project (ban akun nakal, hapus project fake).

Monitoring transaksi & laporan.

Analogi: satpam kampus → ngejaga supaya ekosistem tetap sehat.
